# ΛKUMΛ 2.1.1

ΛKUMΛ/Informatyk to w 100% darmowy zbiór testów dla  Technika Informatyka, koncentrujący się na kwalifikacjach INF-03/EE.09 oraz INF-02/EE.08. Aplikacja została stworzona, aby pomóc uczniom skutecznie przygotować się do egzaminów.

## Funkcje

- **Kompleksowy Zbiór Pytań egzaminacyjnych**: Szeroka gama pytań  obejmujących 99% zagadnień w ramach kwalifikacji INF-03/EE.09 oraz INF-02/EE.08.
- **Różnorodne Wersje Nauki**: Ucz sie w trybie 40 pytań jak na egzaminie lub w trybie 1 pytanie dla szybszego efektu.
- **Ogólno Dostępna baza pytań**: Jako jedyny projekt posiada całkowicie darmową i w pełni otwartą bazę pytań do wglądu dla każdego użytkownika.
- **Archiwum Arkuszy**: Projekt posiada we współpracy z arkusze.pl liste wszystkich arkuszy z lat ubiegłych do pobrania za darmo.
- **Interfejs Internetowy**: Nowoczesny i intuicyjny interfejs zbudowany w oparciu o TailwindCSS.
- **Regularne Aktualizacje**: Treści są aktualizowane, aby odzwierciedlać najnowsze standardy i tematy egzaminacyjne.
- **Kalkulatory**: Wszelkie niezbędne kalkulacje które mogą przydać się na egzaminach systemy liczbowe,adresy ip,kolory

## Instalacja

Aby uruchomić projekt:

1. **Sklonuj repozytorium**:

   ```bash
   git clone https://github.com/kaxuri/informatyk.git
   ```

2. **Przejdź do katalogu projektu**:

   ```bash
   cd informatyk
   ```

3. **Zainstaluj zależności projektu**:

   ```bash
   npm installhttps://github.com/kaxuri/informatyk/blob/main/README.md
   ```

4. **Uruchom aplikację**:

   ```bash
   npm run dev
   ```

5. **Otwórz aplikację**:

   Otwórz przeglądarkę i przejdź do `http://localhost:3000`, aby zobaczyć aplikację.

## Użytkowanie

Po uruchomieniu aplikacji możesz przeglądać dostępne testy  i wybrać dowolny test, aby rozpocząć ćwiczenia. Interfejs jest intuicyjny i pozwala na łatwe poruszanie się między różnymi testami i modułami.
